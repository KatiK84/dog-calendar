# Календарь собак (PWA)

Это простой PWA-календарь “лента” для двух собак (Бонни и Нола).
Данные хранятся локально в браузере (localStorage).

## Функции
- Лента: Ближайшие / Просрочено / История
- Фильтр: Все / Бонни / Нола
- Добавить/редактировать события
- Отметить “Сделано” / снять отметку
- Перенести дату
- “Добавить напоминание” — создаёт .ics файл календаря с напоминанием за 7 дней

## Запуск локально
1) Открой терминал в этой папке
2) Запусти простой сервер (любой вариант):
   - Python: `python -m http.server 5173`
   - Node: `npx serve .`

3) Открой в браузере:
   - http://localhost:5173

## Установка как приложение
- Chrome/Edge: меню → “Установить приложение”
- iPhone (Safari): “Поделиться” → “На экран «Домой»”

> Примечание: PWA хорошо устанавливается на HTTPS. На localhost обычно тоже ок.
